Infrared Small Target Detection

Decoupling Polarity with Self-Prompts

SPD is a polarity-aware segmentation framework for dim infrared small targets whose appearance may switch between bright and dark under complex backgrounds, low signal-to-clutter ratios, and polarity reversal.

Polarity-aware self-prompt detection

SPD combines re-parameterized infrared feature extraction, adaptive edge denoising, and bright/dark prompt tokens to separate target polarity during mask decoding.

Rep-EFEM encoder

Multi-stage infrared feature extraction with deploy-time branch fusion for efficient inference.

Adaptive edge denoising

A learnable Sobel/PDE-inspired edge path strengthens weak target boundaries while suppressing clutter responses.

Self-prompt decoder

Bright and dark learnable tokens guide feature selection and decouple opposite target appearances.

Quick start after release

Commands mirror the reproducibility scripts in the repository and can be updated with final checkpoint paths before publication.

pip install -r requirements.txt
python train.py --config configs/spd_irtiny_bd_10k.yaml
python test.py --config configs/spd_irtiny_bd_10k.yaml --checkpoint work_dirs/spd_irtiny_bd_10k/best.pth --save-pred
python inference.py --config configs/spd_mydata.yaml --checkpoint work_dirs/spd_mydata/best.pth --img-dir demo/images --out-dir demo/predictions
